Brenda Enzyme
=============

Creating a dataset from Brenda using the [brendapy parser](https://github.com/matthiaskoenig/brendapy)

Usage
-----

In the terminal

```
list_parameter = []
path_file_brenda =
brenda_en_code -m list_parameter path_file_brenda
```

list of possible parameters :
* KM
* ...
